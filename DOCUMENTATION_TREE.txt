=============================
Problem: "Wrong Name" error when creating test_deneme.xml

Solution: File names must follow C variable naming rules
   ❌ test-deneme.xml  (hyphen not allowed)
   ✅ test_deneme.xml  (underscore is OK)

   or VISUAL_NAMING_GUIDE.md (Visual)
