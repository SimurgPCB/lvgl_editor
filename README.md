## ğŸ”‘ Key Concepts

### File Naming Rules (CRITICAL!)

âœ… **ALLOWED:**
- Letters: `a-z`, `A-Z`
- Numbers: `0-9` (not at start)
- Underscores: `_`

âŒ **NOT ALLOWED:**
- Hyphens: `-`
- Dots: `.` (except `.xml` extension)
- Spaces: ` `
- Special chars: `@#$%^&*()`

**Examples:**
```
âœ… test_deneme.xml       (underscore)
âœ… my_button.xml         (underscore)
âŒ test-deneme.xml       (hyphen)
âŒ my.button.xml         (dot)
```

### Reference Syntax

```xml
<!-- Constants (from globals.xml) -->
#unit_md
#primary_color
#icon_home

<!-- Component properties (from API) -->
$title
$bg_color
$icon

<!-- Data binding (from subjects) -->

bind_checked="is_on"
```
